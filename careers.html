🎯 GATE CS Exam Pattern

  • Paper Code CS (Computer Science & IT)
  • Total Marks 100 marks, 65 questions
  • Duration 3 hours
  • Sections GA (15%) + CS Core (85%)
  • Question Types MCQ + MSQ + NAT
  • Negative Marking 1/3 for MCQ only

📖 GATE CS Key Topics

  • High Weight Algorithms, DBMS, OS
  • High Weight Data Structures, TOC
  • Medium CN, COA, Compiler Design
  • Medium Digital Logic, Programming
  • Low Discrete Math, Linear Algebra
  • GA Verbal + Numerical Aptitude

Complete Preparation Roadmap for CSE Placements
A structured, semester-wise plan to maximize your chances across all opportunity types.

📅 Year-wise Timeline (B.Tech)

  • Year 1–2 Master C++/Python, start competitive programming
  • Year 2 (Jul–Dec) DSA deeply: arrays, trees, graphs, DP
  • Year 2 (Jan–Jun) Apply for summer internships
  • Year 3 (Jul–Sep) Build 2–3 solid projects; LeetCode 300+
  • Year 3 (Oct–Dec) Off-campus drives start – apply aggressively
  • Year 4 (Jan–Jun) Final year drives; GATE prep if interested

💻 Must-Know CS Subjects

  • DSA Arrays, Trees, Graphs, DP, Heaps, Tries
  • OOPs Inheritance, Polymorphism, Design Patterns
  • DBMS SQL, Normalization, Indexing, Transactions
  • OS Scheduling, Memory Mgmt, Deadlock, Sync
  • CN TCP/IP, DNS, HTTP, OSI, Routing
  • System Design LLD + HLD (for product roles)
🗺️ Off-Campus Placement Step-by-Step
1
Build a Strong Foundation (Months 1–4)

Complete Striver's A2Z DSA Sheet (450+ problems). Solve 150+ LeetCode problems across Easy/Medium/Hard. Learn one language deeply — Java or C++. Maintain GitHub with at least 3 visible projects.

2
Register on All Major Portals (Immediately)

Create profiles on: TCS NextStep, Infosys InfyTQ, Wipro Careers, Cognizant Careers, LinkedIn, Naukri, Internshala, Unstop. Set job alerts with keywords "SDE fresher", "CSE 2026 batch", "software engineer trainee".

3
Crack Aptitude & Online Tests (Month 3–5)

Practice TCS, Infosys, Wipro, Accenture-specific aptitude on PrepInsta and IndiaBIX. Most off-campus drives start with a 2–3 hour online assessment covering aptitude, logical reasoning, and 2–3 coding questions. Prepare for all three sections simultaneously.

4
Apply Actively (August – December is peak season)

Aug–Dec is when 80% of off-campus drives open. Apply to at least 20–30 companies. Don't wait for the perfect company — cast a wide net. Join Telegram groups that post daily alerts. Track applications in a spreadsheet.

5
Ace the Technical Interview

Practice 2–3 mock interviews per week on Pramp, Interviewing.io, or with peers. Revise all CS fundamentals (OS, DBMS, CN, OOPs). Be ready to explain your projects end-to-end — technologies used, your role, challenges, and learnings.

6
For GATE / Govt Jobs (Parallel Track)

Start GATE preparation from Year 3. Focus on GATE CS syllabus: Algorithms, DBMS, OS, TOC, CO&A, Discrete Math, CN. Use NPTEL lectures + PYQs. A GATE score above 650 opens NIC, DRDO, IOCL, ONGC, BARC simultaneously.

7
Keep Competing in Hackathons

Participate in SIH, TCS CodeVita, HackWithInfy, and at least one LeetCode/Codeforces contest every week. Contest ratings and hackathon wins are strong differentiators in off-campus resumes. Even reaching the finals of SIH is worth mentioning.

📄 Resume Checklist

  • Length Strictly 1 page for freshers
  • Projects 2–3 with tech stack + impact metrics
  • CP Links Codeforces/LeetCode profile with rating
  • GitHub Active repos with README files
  • Certifications AWS, GCP, NPTEL, Coursera courses
  • Skills Languages, frameworks, tools — no fluff

  • ❌ Skipping HR prep — culture fit matters
  • ❌ Only relying on college placement cell
